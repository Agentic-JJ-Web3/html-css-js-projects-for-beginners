📖 Basic Functions Guide

AC (All Clear) Resets calculator completely
CE (Clear Entry) Clears current number only
⌫ (Backspace) Deletes last digit entered
MC (Memory Clear) Clears stored memory value
MR (Memory Recall) Shows stored memory value
M+ (Memory Add) Adds current number to memory
M- (Memory Subtract) Subtracts current number from memory
% (Percentage) Calculates percentage of first number
